    a. npm install -D tailwindcss postcss autoprefixer

**
    content:[

    ],**

    c. Add tailwind Directives to your CSS   -> In`./src/index.css`

    **@tailwind** base;

    @tailwind** components;

    @tailwind** utilities;

4. npm run dev
